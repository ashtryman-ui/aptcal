# 아파트 재건축 입주 현금 계산기

재건축 조합원이 입주 시 실제로 준비해야 할 현금을 항목별로 계산하는 모바일 웹앱입니다.

---

## 주요 기능

### 조합원
- 이름 + 휴대폰번호로 로그인
- 보유 호수별 탭 전환
- 입주 현금 계산기
  - 추가분담금 (평당 1,055만원 기준)
  - 취득세 자동계산 (6억 이하 / 6~9억 / 9억 초과 구간별)
  - 인지세 · 법무사 수수료 자동계산
  - 이주비 대출 이자
  - 중도금 대출 이자
  - 옵션 · 선수관리비 · 이사비 · 기타
  - 잔금 담보대출 차감
- 계산 결과 저장 및 대시보드 요약 조회

### 관리자
- 전용 비밀번호로 로그인
- 조합원 로그인 기록 조회 (날짜별 · 사람별 방문기록)
- 본인 호수 계산기 이용 가능

---

## 화면 구성

```
index.html        로그인 페이지
dashboard.html    대시보드 (호수별 탭 · 저장 요약)
calculator.html   입주 현금 계산기
logs.html         로그인 기록 (관리자 전용)
members.js        공통 데이터 · 유틸리티
```

## 기술 스택

| 구분 | 사용 기술 |
|---|---|
| UI 프레임워크 | Bootstrap 5.3.3 |
| 아이콘 | Font Awesome 6.5.2 |
| 데이터 저장 | localStorage |
| 배포 | Vercel (프론트엔드) |

> 현재 버전은 서버 없이 `localStorage`를 사용합니다.  
> 추후 Render(백엔드) + PostgreSQL(DB) 연동으로 확장 예정입니다.

## 취득세 계산 기준

| 분양가 | 세율 |
|---|---|
| 6억 이하 | 1.1% |
| 6억 초과 ~ 9억 이하 | 구간 비례 (1.1% ~ 3.3%) |

## 로컬 실행

별도 빌드 없이 바로 실행 가능합니다.

```bash
# 간단한 로컬 서버 실행 (Python)
python3 -m http.server 8080

# 브라우저에서 접속
http://localhost:8080
```
